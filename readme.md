# YTDOWNLOAD
An original Youtube downloader
I made this originally for myself to download a full channel's playlist (ever heard of tyler vitelli?)
Here are the features:

* Custom File Names
* No Ads
* SponsorBlock
* Auto remove videos with numbers at start (made this for tyler vitelli lol)
* Force IPV4
* Sound when download finished
* Auto retry
* Shutdown when finished
* Auth Bypass
* Speed limit
* Max file size
* Subtitles as SRT
* Save description as TXT
* Burn subtitles into the video (basically, it displays the subtitles on the actual video MP4)
* Keep video when extracting audio
* File type (FLAC MP3 MP4 WAV)
* quality (4K 1440P 1080P 720P)
* Embed thumbnail
* embed chapters/tags
* Playlist index targeting (start/end video numbers)


Did i mention; All of these can be turned on or off and it has a mass link processer?
Lastly, you can download videos, shorts, playlists, or the entire channel! (or the channel's shorts or playlists)
